WEB DEVELOPMENT
Web development is everything involved in the creation of a website. Typically it refers to the coding and programming side of web site production as opposed to the web design side. It encompasses everything from a simple page of HTML text to complex, feature-rich applications designed to be accessed from various Internet-connected devices. Examples of feature-rich web development include ecommerce websites, content management systems (CMS) and social networks. Common web development programming languages and software include Hypertext Markup Language (HTML), Cascading Style Sheets (CSS), JavaScript, PHP, Drupal and MySQL.
